This repository contains a data analytics project focusing on Customer Segmentation using Recency, Frequency, and Monetary (RFM) analysis on retail transactional data.

## Project Structure
- `retail.csv`: The raw transactional dataset.
- `Retail_Customer_Analytics.ipynb`: The main Jupyter Notebook containing the data cleaning, EDA, RFM calculation, and business insights.

## Business Context
An online retail store wants to understand its customer base better to create targeted marketing campaigns. Treating all customers the same leads to inefficient marketing spend and lower conversion rates.

**The goals of this project are:**
1. Clean and preprocess messy transactional data.
2. Calculate Recency, Frequency, and Monetary (RFM) metrics for every customer.
3. Segment customers into actionable groups (e.g., Champions, At-Risk, Lost) to drive personalized marketing strategies.

## Key Technical Skills Demonstrated
- Data Cleaning (Handling Missing Values, Duplicates, Bad Data)
- Feature Engineering (Calculating Revenue, Datetime manipulation)
- Data Aggregation (Creating Customer-level metrics from Transaction-level data)
- Customer Segmentation / RFM Analysis
- Data Visualization (Matplotlib, Seaborn)

## How to Run
1. Clone this repository.
2. Ensure you have Python installed with the following libraries: `pandas`, `numpy`, `matplotlib`, `seaborn`.
3. Open `Retail_Customer_Analytics.ipynb` in Jupyter Notebook or Google Colab and run all cells.
