1. Marketing Strategy & Foundational Metrics

Treating all customers uniformly leads to sub-optimal marketing spend and lower retention. This project processes transactional data to evaluate three core behavioral vectors: Recency (days since last purchase), Frequency (total transactions completed), and Monetary Value (aggregate revenue contribution).

2. Engineering the Data Processing Pipeline

The preprocessing script cleans raw transaction lines using clear, reproducible steps:

# Calculate baseline customer transaction parameters
customer_matrix = df.groupby('CustomerID').agg({
    'InvoiceDate': lambda x: (snapshot_date - x.max()).days,
    'InvoiceNo': 'count',
